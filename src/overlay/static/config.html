How to set up ShimaTTS
  1. Login with Twitch - your channel name fills in automatically.
  2. Create a channel point reward on Twitch: Creator Dashboard → Viewer Rewards → Channel Points → Manage Rewards & Challenges → Add New Custom Reward. Turn on "Require Viewer to Enter Text" - that text is what gets spoken. Then enter the exact same reward name in this app.
  3. Upload a voice sample - 10 to 30 seconds of clear speech (WAV or MP3), no music or background noise. Select it in the Voice library.
  4. Upload an alert GIF and select it in the Overlay library.
  5. Add the overlay to OBS: Sources → + → Browser, URL http://localhost:7878/overlay, Width 1260, Height 260, and enable "Refresh browser when scene becomes active". The alert fills the source exactly - drag it wherever you want the alert to appear. Background stays transparent between alerts.
  6. Hit Save & Start, then redeem your reward on stream to test!
